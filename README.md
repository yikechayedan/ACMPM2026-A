# ACMPM2026-A
## 2026-6-13 数据预处理
系统性错误：2025-2-7到8日记录错误：CLR与CL2写反
            2025-7-22 7-13点：R/W NTU与R/W CLR写反

特征筛选：为避免引入与目标变量同环节的伴随状态变量造成解释偏差，候选特征不纳入 PH、CLR、CL2；泵工作状态由R/W FLOW直接表示，候选特征不纳入R/W FLOW；F/RIDE缺失比例太高，因此仅由ALUM来表示矾投加量，候选特征不纳入F/RIDE；对remark：2025-12-15 7：00与2025-3-4 23：00，FILT.NTU表现为异常高值，根据备注可知这两日对过滤装置进行了Back wash。但该行为对FILT.NTU的作用效果不稳定，可能与当日过滤装置的清洁程度有关，因此对这两个异常点进行处理，将REMARK剔除

异常点筛选：考虑到多项指标具有明显的季节性特征，存在连续高值和连续低值的现象，因此使用rolling-robust-z找出潜在异常，人工筛选潜在异常点，以便将正常工况突变和测量错误区分开来。

处理方式：置为缺失再线性插值

插值：剩下的特征里，由于长缺失的只有R/W PH和ALUM，它们在全年范围内都保持在一个稳定的范围，因此使用前向填充，其他缺失则使用线性插值
